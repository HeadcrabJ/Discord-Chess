### Redirect `How to add?`

```html

<div id='redirect'>
	<span id='redirect-path'><!-- JSON key from info.js --></span>
	<!-- !Important -->
	<script src='info.js'></script>
	<script src='functions/redirect.js'></script>
</div>
```

### Localization `How to localize?`

```html
<span localize='key'></span><!-- key - is JSON key from info.js file -->
```
